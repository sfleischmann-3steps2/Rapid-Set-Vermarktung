## 1. Gesamtprojekt-Überblick

### Was ist das Projekt?

Eine saisonale B2B-Vermarktungskampagne für **Rapid Set ASPHALT REPAIR MIX (ARM)** von KORODUR -- ein zementbasiertes, polymervergütetes Reparaturmaterial für Asphaltflächen. ARM schließt die Lücke zwischen Kaltmischgut (hält nicht) und Heißasphalt (zu aufwändig). Saisonaler Aufhänger: Frostschäden im Frühjahr.

## 2. Kampagnenkonzept

### 2.1 Zielsetzung

**Primärziel:** Neue Kunden für ARM gewinnen -- Erstkäufe generieren durch ein niedrigschwelliges Testangebot.

**Sekundärziel:** ARM bei Tiefbauämtern, GaLaBau-Betrieben und Facility Managern als "die bessere Alternative zu Kaltmischgut" bekannt machen.

**Tertiärziel:** Leads qualifizieren und in den Vertriebsprozess (CRM) überführen.

**Messbare Ziele:**
- Kontaktanfragen über Landingpage (Formular + Telefon)
- Datenblatt-Downloads (Micro-Conversion)
- Telefonisch qualifizierte Leads → Angebote → Bestellungen
- E-Mail-Öffnungs- und Klickrate

### 2.2 Zielgruppen

| Priorität | Zielgruppe | Warum ARM? | Leads vorhanden |
|-----------|-----------|------------|-----------------|
| **1** | **Kommunen / Tiefbauämter / Bauhöfe** | Frostschäden jedes Jahr, knappes Budget, Bürgerbeschwerden | 1.081 (Tiefbau) |
| **2** | **GaLaBau-Betriebe** | Wege, Einfahrten, Terrassen reparieren -- Kundenzufriedenheit | 728 (GaLaBau) |
| **3** | **Facility Management / Industrieunternehmen** | Parkplätze, Zufahrten -- Haftungsrisiko, Erscheinungsbild | Via LP + Ads |
| **4** | **Verarbeiter / Handwerker** | Einfache Verarbeitung, zufriedene Auftraggeber | Via LP + Ads |

### 2.3 Das Angebot

| | |
|---|---|
| **Aktionsangebot** | 2 Paletten kaufen (2 x 48 Sack) → **24 Sack ohne Berechnung** |
| **Kommunikation** | "24 Sack geschenkt" (NICHT als "Rabatt" kommunizieren) |
| **Laufzeit** | Bis **Ende März 2026** (intern: Option auf Ende April) |
| **Einlösung** | Pro Kunde einmalig |
| **Positionierung** | "Risikofrei testen" -- genug Material für einen echten Praxistest |

### 2.4 Storyline (roter Faden durch ALLE Kanäle)

Die Kampagne folgt einer klaren 4-Schritt-Dramaturgie:

```
1. PROBLEM     → Frostschäden sind akut. Straßen, Wege, Parkplätze sind kaputt.
2. FRUSTRATION → Kaltmischgut hält nicht. Heißasphalt ist zu aufwändig.
3. LÖSUNG      → ARM: Schnell, einfach, dauerhaft. Sack auf, Wasser dazu, fertig.
4. AKTION      → Jetzt risikofrei testen: 24 Sack geschenkt. Jetzt bei uns melden.
```

**In einem Satz:**
> "Frostschäden sind da -- ARM repariert sie schnell und dauerhaft. Testen Sie jetzt mit 24 Sack geschenkt."

## 3. Landingpage -- Strukturoptimierung

### 3.1 Kernproblem der aktuellen LP-Struktur

Die aktuelle Landingpage (im Marketing-Repo) ist als **allgemeine Produktseite** aufgebaut. Sie erklärt ARM hervorragend, aber:

- **Kein Bezug zur Frühjahrsaktion** -- das Aktionsangebot (24 Sack geschenkt) fehlt komplett
- **Kein saisonaler Aufhänger** -- "Frostschäden sind jetzt akut" wird nicht als Einstieg genutzt
- **CTA ist zu allgemein** -- "Jetzt beraten lassen" statt "Jetzt Aktionsangebot sichern"
- **Zu viel Produkt-Erklärung, zu wenig Dringlichkeit** -- die Seite liest sich wie ein Datenblatt, nicht wie eine Kampagne

### 3.2 Empfohlene Storyline der Landingpage

Die LP soll der Kampagnen-Storyline folgen:

```
HERO:           Frostschäden? → ARM repariert dauerhaft → AKTIONSANGEBOT
PROBLEM:        Warum Kaltmischgut nicht reicht
LÖSUNG:         So einfach geht's (4 Schritte)
BEWEIS:         Referenzprojekte (es funktioniert wirklich)
ANGEBOT:        Frühjahrsaktion: 24 Sack geschenkt
CTA:            Jetzt melden (Formular / Telefon)
```
